The DNA Analyzer

The DNA Analyzer is command line tool to perform a variety of RCA / Log Analyzer, Various application related data collection from postgres database table, Finding inconsistent data existence from various application related postgres table and Search Known defect based on Log Analyzer result. This tool uses various CLI Show command in DNAC, Show command in postgres tables and Makes REST API calls to CDETS Application. This toool is completely read only and it will not perform any write operation into DNAC Application.
